\# CodeAlpha Cybersecurity Internship - Task 2



\## Project: Phishing Awareness Training \& Interactive Assessment

This project is an interactive educational module designed to train users on identifying phishing attacks. It combines a high-fidelity Cyberpunk-themed presentation with a live, time-restricted assessment to test user knowledge in real-time.



\### Features:

\- \*\*Interactive Cyberpunk Presentation:\*\* High-quality visual slides designed with Neon Cyan and Pink themes to engage users.

\- \*\*Live Assessment Module:\*\* Integrated Google Form for real-time testing.

\- \*\*Automated Timer System:\*\* Custom Google Apps Script (JavaScript-based) to enforce a 10-minute time limit and prevent late submissions.

\- \*\*Seamless QR Integration:\*\* A custom-designed, neon-styled QR code for instant mobile access to the training quiz.

\- \*\*Security Protocols:\*\* Restricted to "Single Response" and "Shuffle Question Order" to ensure assessment integrity.



\### Repository Contents:

\- \*\*Phishing-Awareness-Training.pptx\*\*: The main interactive presentation file.

\- \*\*Phishing-Quiz-Module-Structure.pdf\*\*: A detailed document outlining the assessment questions and logic.

\- \*\*Phishing-Assessment-QR.png\*\*: The high-resolution QR code image used in the presentation.

\- \*\*Screenshots/\*\*: Visual proof of the presentation slides and the live Google Form interface.



\### How to Use:

1\. Open the \*\*Phishing-Awareness-Training.pptx\*\* file.

2\. Navigate to \*\*Slide 9\*\* to view the "Ultimate Phishing Challenge."

3\. Scan the \*\*QR Code\*\* using a smartphone to launch the live assessment.

4\. Complete the 10-question quiz within the \*\*10-minute limit\*\* enforced by the backend script.
