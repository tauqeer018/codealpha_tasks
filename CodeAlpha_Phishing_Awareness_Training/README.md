This project is an interactive educational module designed to train users on identifying phishing attacks. It combines a **high-fidelity Cyberpunk-themed** presentation with a live, time-restricted assessment to test user knowledge in real-time.

---

## 🚀 Key Features

* **🎨 Interactive Cyberpunk Presentation:** High-quality visual slides designed with Neon Cyan and Pink themes to engage users.
* **📝 Live Assessment Module:** Integrated Google Form challenge for real-time knowledge testing.
* **⏱️ Automated Timer System:** Custom **Google Apps Script (JavaScript)** backend to enforce a strict **10-minute time limit** and prevent late submissions.
* **📲 Seamless QR Integration:** A custom-designed, neon-styled QR code for instant mobile access to the training quiz.
* **🔒 Security Protocols:** Configured with "Single Response" locks and "Shuffle Question Order" to ensure assessment integrity.

## 🛠️ How to Use

1.  **Launch:** Open the `Phishing-Awareness-Training.pptx` file.
2.  **Navigate:** Go to **Slide 9** to view the **"Ultimate Phishing Challenge."**
3.  **Scan:** Use a smartphone to scan the **Neon QR Code** and launch the assessment.
4.  **Complete:** Answer the 10-question quiz within the **10-minute window** tracked by the backend script.
